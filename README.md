# Juego de Ahorcado en JAVA

Este es mi primer juego, en esta oportunidad elegí el clásico "ahorcado" ya que permite aprender varios conceptos de JAVA que son vitales para entender el lenguaje en sí.

Reglas: Escribir letra por letra en consola, hasta completar la palabra. Los intentos son limitados.

# Herramientas
1. Clase Scanner.
2. Declaraciones y asignaciones.
3. Arreglos.
4. Estructuras de control iterativas.
5. Strings y métodos valueOf, lenght, charAt, toLowerCase.
6. Condicionales.

# Hangman game in JAVA

This is my first game, this time I chose the classic "hangman" because it allows you to learn several JAVA concepts that are vital to understanding the language itself.

Rules: Type letter by letter into the console until you complete the word. Attempts are limited.

# Tools
1. The Scanner class. 2. Declarations and assignments. 3. Arrays. 4. Iterative control structures. 5. Strings and the methods valueOf, length, charAt, toLowerCase. 6. Conditionals .
