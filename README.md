# Juego de Ahorcado en JAVA

Este es mi primer juego, en esta oportunidad elegí el clásico "ahorcado" ya que permite aprender varios conceptos de JAVA que son vitales para entender el lenguaje en sí.

Reglas: Escribir letra por letra en consola, hasta completar la palabra. Los intentos son limitados.

# Herramientas
1. Clase Scanner.
2. Declaraciones y asignaciones.
3. Arreglos.
4. Estructuras de control iterativas.
5. Strings y métodos valueOf, lenght, charAt, toLowerCase.
6. Condicionales.
